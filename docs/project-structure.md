# Rune Drift Survivors Project Structure

## Runtime Code

- `src/main.jsx` - current playable prototype. Game loop, combat, map, HUD, and rendering live here for now.
- `src/config/assets.js` - active model URL manifest and preload list.
- `src/styles.css` - HUD, modal, and shell styles.
- `vite.config.js` - build configuration, including vendor chunk splitting for React, R3F, Three, and effects.
- `scripts/gltf-to-glb.mjs` - converts archived `.gltf` source models into runtime `.glb` files under `public/models`.

## Runtime Assets

- `public/models/` contains only assets loaded by the current game.
- `public/models/nature-kit/` contains the active open-field environment props from Kenney Nature Kit.

## Source And Archive Assets

- `assets/source/kenney-nature-kit/` keeps the downloaded Kenney Nature Kit source package.
- `assets/archive/public-models/` keeps older experiment assets removed from `public` so Vite no longer copies them into `dist`.
- `assets/archive/public-models/gltf-source/` keeps source `.gltf` exports after runtime copies are converted to `.glb`.
- `assets/archive/public-textures/` keeps old standalone textures that are not needed by the current runtime.

## Generated Folders

- `dist/` is generated by `npm run build`.
- `output/` contains Playwright screenshots and verification artifacts.
- `.playwright-cli/` contains local browser automation logs.

These generated folders are ignored by git.
